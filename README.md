# Jazmín | Web Developer Jr - About Me

Proyecto personal desarrollado con **React** y **Vite**, enfocado en prácticas profesionales de arquitectura de componentes y manejo de datos asíncronos.

## Tecnologías y Prácticas
- **React 18**: Uso de Hooks (`useState`, `useEffect`).
- **Axios**: Comunicación con APIs externas.
- **JSON Server**: Simulación de base de datos REST para contenido dinámico.
- **Git Flow**: Manejo de ramas (`feature branches`) y **Conventional Commits**.
- **CSS3**: Diseño responsivo con Variables, Flexbox.

##  Instalación y Uso

1. Clona el repositorio:
   `git clone https://github.com/tu-usuario/about-me.git`
2. Instala las dependencias:
   `npm install`
3. En una terminal, inicia el servidor de datos (Mock API):
   `npm run server`
4. En otra terminal, inicia la aplicación:
   `npm run dev`
